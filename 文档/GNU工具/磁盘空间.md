# 磁盘空间 (df)

> **对联系统理念**: 以日常汉语为导向，精准表意，规范统一。

## 📋 核心定义

**磁盘空间** 在汉语编程中对应英文命令：**df**。

显示文件系统磁盘空间的使用情况，包括总容量、已用和剩余空间。

## 💻 用法举例

### 基础用法

| 语境 | 命令示例 |
| :--- | :--- |
| **中文** | `磁盘空间 -h` |
| **英文** | `df -h` |

### 参数详解

| 参数 | 示例 | 英文参数 | 作用说明 |
| :--- | :--- | :--- | :--- |
| **人类可读** | `磁盘空间 人类可读` | `-h` / `--human-readable` | 以人类可读的格式显示容量（如1K、234M、2G），便于理解。 |
| **所有文件系统** | `磁盘空间 所有` | `-a` / `--all` | 显示所有文件系统，包括伪文件系统和重复项。 |
| **指定类型** | `磁盘空间 类型 ext4` | `-t [类型]` / `--type=[类型]` | 仅显示指定类型的文件系统，如ext4、ntfs等。 |
| **排除类型** | `磁盘空间 排除类型 tmpfs` | `-x [类型]` / `--exclude-type=[类型]` | 排除指定类型的文件系统不显示。 |
| **本地文件系统** | `磁盘空间 本地` | `-l` / `--local` | 仅显示本地文件系统，不包括网络挂载。 |
| ** inode信息** | `磁盘空间 inode信息` | `-i` / `--inodes` | 显示inode使用情况而非块使用情况。 |
| **总用量** | `磁盘空间 总用量` | `--total` | 在所有输出最后显示一个总计行。 |
| **帮助信息** | `磁盘空间 帮助信息` | `--help` | 显示完整的帮助文档和所有可用参数。 |
| **版本信息** | `磁盘空间 版本信息` | `--version` | 显示命令的版本号并退出。 |
