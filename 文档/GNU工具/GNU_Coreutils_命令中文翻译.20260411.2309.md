# GNU Coreutils 命令中文翻译手册

> **说明**: 本文档提供了 GNU Coreutils 所有命令的中英文对照翻译，便于理解和使用。
>
> ### 💡 对联系统核心理念
> 1. **规范统一**：语言可以不一样，但只要规范一致，认知成本就能降到最低。
> 2. **日常导向**：API、ABI、接口及命名均以**日常汉语使用习惯**为导向，摒弃晦涩的技术实现术语。
> 3. **精准表意**：追求“望文生义”，让编程指令像日常对话一样自然流畅。
> 4. **中英对照**：以中文为主，英文为辅。中文是核心逻辑的表达，英文作为技术实现的辅助参考。
>
> **格式**: `目录名 = 中文名 (功能说明)`

## 📌 特别说明

### 别名支持

- **`[`** → `test` 的别名（用于条件测试）
- **`dir`** → `ls` 的别名（以列格式显示）
- **`vdir`** → `ls` 的别名（以详细格式显示）

这些别名在 `uucore` 中通过 `get_canonical_util_name()` 函数进行映射。
