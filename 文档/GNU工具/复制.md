# 复制 (cp)

> **对联系统理念**: 以日常汉语为导向，精准表意，规范统一。

## 📋 核心定义

**复制** 在汉语编程中对应英文命令：**cp**。

将源文件或文件夹复制到目标位置，生成一个完全相同的副本。

## 💻 用法举例

### 基础用法

| 语境 | 命令示例 |
| :--- | :--- |
| **中文** | `复制 -r 源文件夹 目标位置` |
| **英文** | `cp -r source_dir dest_dir` |

### 参数详解

| 参数 | 示例 | 英文参数 | 作用说明 |
| :--- | :--- | :--- | :--- |
| **递归复制** | `复制 递归 源文件夹/ 目标文件夹/` | `-r` / `--recursive` | 递归复制目录及其所有内容，包括子目录。 |
| **保留属性** | `复制 保留属性 程序.sh /usr/local/bin/` | `-p` / `--preserve` | 保留文件的权限、时间戳、所有者等属性。 |
| **交互式** | `复制 交互 报告.docx 备份/` | `-i` / `--interactive` | 覆盖现有文件前询问确认。 |
| **更新复制** | `复制 更新 源代码.c 编译目录/` | `-u` / `--update` | 仅当源文件比目标文件新时才复制。 |
| **详细输出** | `复制 详细 大型文件.iso 备份/` | `-v` / `--verbose` | 显示正在复制的每个文件名称。 |
| **帮助信息** | `复制 帮助信息` | `--help` | 显示完整的帮助文档和所有可用参数。 |
| **版本信息** | `复制 版本信息` | `--version` | 显示命令的版本号并退出。 |

## ✅ 执行结果介绍

在目标位置产生一个与原文件一模一样的副本。
