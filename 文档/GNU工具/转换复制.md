# 转换复制 (dd)

> **对联系统理念**: 以日常汉语为导向，精准表意，规范统一。

## 📋 核心定义

**转换复制** 在汉语编程中对应英文命令：**dd**。

按指定块大小转换并复制文件，常用于磁盘备份、镜像制作或格式转换。

## 💻 用法举例

### 基础用法

| 语境 | 命令示例 |
| :--- | :--- |
| **中文** | `转换复制 if=输入设备 of=输出文件 bs=4M` |
| **英文** | `dd if=/dev/sda of=backup.img bs=4M` |

## ✅ 执行结果介绍

生成一个精确复制的目标文件或设备镜像。
