# 查看大小 (du)

> **对联系统理念**: 以日常汉语为导向，精准表意，规范统一。

## 📋 核心定义

**查看大小** 在汉语编程中对应英文命令：**du**。

估算文件或文件夹在磁盘中实际占用的存储空间，帮助分析存储占用情况。

## 💻 用法举例

### 基础用法

| 语境 | 命令示例 |
| :--- | :--- |
| **中文** | `查看大小 人类可读 报告.pdf` |
| **英文** | `du -h report.pdf` |

### 参数详解

| 参数 | 示例 | 英文参数 | 作用说明 |
| :--- | :--- | :--- | :--- |
| **人类可读** | `查看大小 人类可读 项目文件夹/` | `-h` / `--human-readable` | 以人类可读的格式显示大小（如K、M、G），便于理解。 |
| **仅总计** | `查看大小 仅总计 数据文件夹/` | `-s` / `--summarize` | 只显示每个参数的总计，不列出子目录。 |
| **最大深度** | `查看大小 最大深度 1 项目文件夹/` | `-d [深度]` / `--max-depth=[N]` | 限制递归显示的目录层级深度。 |
| **排除文件** | `查看大小 排除文件 "*.tmp" 工作目录/` | `--exclude=[模式]` | 排除匹配指定模式的文件和目录。 |
| **总计** | `查看大小 总计 文档/ 下载/` | `-c` / `--total` | 在所有输出最后显示一个总计行。 |
| **块大小** | `查看大小 块大小 1M 大型数据库.db` | `-B [大小]` / `--block-size=[大小]` | 使用指定的块大小单位显示。 |
| **帮助信息** | `查看大小 帮助信息` | `--help` | 显示完整的帮助文档和所有可用参数。 |
| **版本信息** | `查看大小 版本信息` | `--version` | 显示命令的版本号并退出。 |

## ✅ 执行结果介绍

输出包含两列：**占用空间** 和 **路径**。
例如：`1.2G    项目文件夹/`。如果使用 `-h` 参数，空间会以 K、M、G 等单位显示。
